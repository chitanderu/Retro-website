# Building This Site

This site is built with some fun tech:

- **Next.js 15** with App Router
- **Tailwind CSS v4** for styling
- **DaisyUI v5** for component primitives
- **Motion** (framer-motion) for animations

The design is inspired by 2000s Japanese personal sites and the Y2K kawaii aesthetic.
